> 공식 페이지: https://www.snowflake.com/snowflake-hackathon-2026-korea/

## 주요 일정

| 일정 | 날짜 |
|------|------|
| 참가 등록 | 2026.03.17 ~ 04.05 |
| 개발 및 제출 | 2026.04.01 ~ 04.12 |

| 본선 행사 | 2026.04.29 (화) 15:30~20:00, 레드버튼 컬쳐스테이지 (강남) |

## 팀 구성

- 팀 규모: 1~4명
- 모든 팀원은 동일한 팀명으로 개별 등록 필수
- 팀 구성 변경은 등록 기간 내에서만 가능
- Snowflake 계정: AWS Seoul 또는 US West (Oregon) 리전 사용

## 기술 요건

- **필수**: Snowflake 플랫폼 핵심 기술로 포함
- **허용 언어**: Python, Java, Scala
- **허용 도구**: Snowpark, Cortex AI, Streamlit, Snowflake Marketplace, Container Services
- **기타**: 오픈소스 라이브러리, 파트너 솔루션 (AWS, H2O.ai, Dataiku) 사용 가능
- **가산점**: Cortex AI, Streamlit, Dynamic Tables, Notebooks 등 Snowflake 기능 활용 시
- 무료 30일 트라이얼 계정 사용 (만료 시 신규 생성)
