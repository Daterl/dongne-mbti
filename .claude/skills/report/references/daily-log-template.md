# Daily Log 템플릿

wiki-writer 에이전트가 Daily Log 페이지를 작성할 때 이 템플릿을 따른다.

```markdown
# Daily Log: {YYYY-MM-DD}

> Phase {N} | Day {D}/5 | 크레딧 사용: ~${사용} / $40

## 완료된 티켓
| 티켓 | 제목 | 산출물 |
|------|------|--------|
| #{N} | {제목} | {파일 경로 또는 설명} |

## 주요 작업
- {작업 1}: {결과 요약}

## 코드 변경
- `{파일 경로}`: {변경 내용 한 줄}

## 발견/이슈
- {이슈}: {대응 또는 TODO}

## 내일 계획
- [ ] {작업}
```
